Lo que construimos

La Aplicación

Autenticación con roles
Registro de pacientes, login para 3 roles distintos (Paciente, Médico, Admin) con JWT y control de acceso por endpoint.
Reserva de turnos
Selección de especialidad → médico → fecha/hora → notas. Validación de fechas futuras y flujo de estados completo.
Panel del Médico
Vista de agenda propia, confirmación de turnos, registro de completados y no-shows. Gestión de disponibilidad semanal.
Panel de Administrador
Dashboard con métricas, gestión completa de médicos, especialidades, sedes, usuarios y log de auditoría.
Perfil de paciente
Edición de datos personales, historial de turnos con filtros, cancelación (ventana 24h) y reprogramación.
API REST documentada
25+ endpoints organizados en módulos, respuesta estándar JSON, manejo de errores y logging completo de requests.
